Используемое ПО

При разработке проекта используется утилита Spin для верификации корректности программных моделей, т.е. проверки заданных программных моделей на выполнение требуемых заказчиком задач. Пакет Spin позволяет:
  • строить модели параллельных программ (протоколов, драйверов, систем логического контроля и управления) и широкого класса дискретных систем
  • выразить требуемые свойства их поведения
  • автоматически проверить выполнение требуемых свойств параллельных систем на их моделях на основе формального подхода.
Spin полностью удовлетворяет нуждам проекта в виду простой формулировки его цели, т.е. верификации модели. Другое ПО, как CADP, CHESS, ISP, Java Pathfinder, используется для более сложных задач.
В проекте также задействован компилятор языка С Cygwin, дополняющий фунцкии Spin, а именно необходимый для преобразования созданной модели в код на языке С и дальнейшей верификации.

Цель проекта

Создать действующую программную модель учебного процесса, построенную на основе структуры факультета высшего учебного заведения, и проверить ее работоспособность, используя формальные методы верификации программ с помощью Spin.

Стадии проекта

  • приобретение необходимого ПО
  • изучение принципов работы ПО
  • составление модели
  • построение модели в Spin
  • преобразование модели в програмный код С
  • верификация модели
  • описание результата
  • защита проекта
